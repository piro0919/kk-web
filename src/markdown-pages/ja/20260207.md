title: "Webサイトでウェイクアップワードを検出する方法"
---

とある業務で、ウェイクアップワードを検出する機能が必要だと言われまして。

ネットで調べたところ、ネイティブの情報は出るものの、Web 系はなかなか厳しそうな感じがしていまして。

ChatGPT や CloudeCode とも壁打ちしまして、一旦それっぽい npm パッケージを作成してみました。

[use-ear](https://github.com/piro0919/use-ear)

---

作り自体は非常にシンプルで、Web Speech API を利用して、マッチングさせているだけという。

あとは画面をつけっぱなしにするとか、音を流しっぱなしにするとか、色々と制約はあるものの、そこまで使用に困ることはないかなぁという感じなのですが。

問題は、[Web Speech API の対応状況](https://developer.mozilla.org/ja/docs/Web/API/Web_Speech_API#%E3%83%96%E3%83%A9%E3%82%A6%E3%82%B6%E3%83%BC%E3%81%AE%E4%BA%92%E6%8F%9B%E6%80%A7) でして、なんと Firefox が対応していないという。

---

そのため、上記のパッケージでは完全対応、とまではなっていないのですが。

概ねモダンな環境では問題なく動くと思います。

Firefox 含め完全対応となってくると、AmiVoice といったリアルタイム文字起こしサービスを併用することになりそうです。

---

そんな感じです。

完全無料で利用できますので、ぜひぜひ。
